== Hands-on Lab Environment

This hands-on lab will be based on at least two OpenShift clusters. The first one will be provided and the second one will be installed from ACM. Both clusters will run on AWS with OpenShift 4.10 Open Environment provided by RHPDS (https://demo.redhat.com)


image::arch/argocd_samrhte2023_arch.png[]

[NOTE]
====
Tested version with:

- Red Hat OpenShift 4.10
- Red Hat Advanced Cluster Management 2.6
- Gitops Operator 1.7
====
